
## 项目环境    
node.js v18.20.4
idea

## 搭建步骤
安装nodejs可用使用nvm来安装也可以直接安装       

npm 配置淘宝源

开启或安装pnpm（二选一）
```shell
corepack enable
```
或者
```shell
npm i -g pnpm
```
选择合适的位置 创建项目        
选项自己选       
my-docs 文件夹参数
```shell
pnpm create vuepress-theme-hope my-docs
```
启动项目
```shell
pnpm docs:dev
```
